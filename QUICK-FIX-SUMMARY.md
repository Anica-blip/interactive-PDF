## 📋 How to Use (Correct Workflow)

### Step 1: Add Button to Asset Library
1. Find "3C Buttons" section (blue gradient)
2. Click button type (e.g., "Generic Button")
3. Enter URL when prompted
4. Button appears in Asset Library below

### Step 2: Add to Page
1. Scroll down to Asset Library
2. Click the button you just added
3. Button appears on your page!

### Step 3: Position & Resize
1. Drag button to desired position
2. Use resize handle (bottom-right corner)
3. Recommended size: 200x80 to 300x120

### Step 4: Generate PDF
1. Click "Generate PDF" button
2. Download your interactive PDF
3. Test the clickable button!

## 💡 Pro Tips

### Button Sizing:
- **Small CTA:** 150x60px
- **Standard:** 200x80px (default)
- **Large Hero:** 300x120px

### Badge Sizing:
- **Small:** 80x80px
- **Standard:** 120x120px (default)
- **Large:** 200x200px

### Best Practices:
- Test URLs before generating PDF
- Leave space around buttons for easy clicking
- Use consistent button sizes throughout
- Position badges in corners or as decorations
